# Duck Club Game

## About 📋

"Duck Club" is a "Club Penguin-inspired" 2D, open exploration game with character customization, interactive minigames, and multiple dynamic scenes created using turtle graphics.

The game utilizes arrow key movement to traverse the world.

### Key Features:

- 3 primary world screens, each with its own interactive gameplay feature
  
- Start area contains a character customization screen, enterable by going to "Feathery Finds," in which the user may change their duck's feather color and accessories. These changes are kept and seen as you travel across the world. *This is primarily achieved through PIL in the Pillow library.*
  
- To the left, there is a section with the duck's igloo/home where, once entered, they can decorate using an assortment of provided furniture. The furniture may be created and deleted with the specified hotkeys (shown in-game) and moved using left click + drag. Changes to the igloo are saved when exiting/re-entering the igloo.
  
- To the right of the start section is a playground in which the user can play "Duck-Jitsu," a rock, paper, scissors-inspired game, by standing on the purple sand pail in the sandbox next to the giant duckling. A quick tutorial is provided inside the game. This minigame also tracks local wins and is displayed when entered.
